# LockPattern
仿支付宝手势密码解锁

## Features
1, 使用 SHA 算法保存手势密码。</br>
2，可以开启震动模式，当选中一个圈的时候，手机会震动。</br>
3，可以开启绘制路径隐藏模式。</br>
